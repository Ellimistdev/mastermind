## Usage
In the current state it must run in irb with:
```bash
irb -r ./mastermind.rb
game = Mastermind.new
```
then you may call the public methods available:
```bash
make_attempt
render
```
eventually you will start the game with
```bash
ruby mastermind.rb
```

## Install
```bash
git clone https://github.com/endotnick/mastermind.git
```
